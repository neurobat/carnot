sumtime
Path: CARNOT/misc

Purpose
This Functions calculates for a given date the sum of seconds, minutes, ... in the year.

Description
The output of this function is real, so on March 6th the sum of year is 0.175, the sum of month is 2.13. This function exists as S-function and as M-script.

Inputs:

  • date [month day hour minute second]

Outputs:

  • sum of [year month day hour minute second]
