wait
Path: CARNOT/basics

Purpose
This block pauses the simulation until the system time reaches the specified date. After the first call, the block pauses the simulation for the specified number of seconds.

Description
When the block appears more than once in a simulink model and each waiting time is e.g. 2 seconds, the function will only wait once 2 seconds per simulation step. If the blocks have different waiting times, the simulation will pause in each simulation step for the longest specified waiting time. For Real Time applications with an interface to the environment which requires the system time instead of the simulation time to evaluate external data.
