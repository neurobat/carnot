economy_ecology
Path: CARNOT/ misc

Purpose
Calculate data for economic and ecological analysis of the systems.


Description
The economy_ecology block provides economic and ecological data depending on the fuel and the used component. This includes the amount of used fuel, fuel costs, thermal and electrical power and exhaust gases. The data is taken from the eco-vector and integrated over the simulation time. All of the values can be displayed when the block is opened. Fuel costs and runtime are written to the workspace and can be used for further calculation with the m-function annualcosts.m.

For information about the Structure see Eco-Vector definition above

Characteristics
Direct Feedthrough   Yes
Sample time                Inherited from driving block
Vectorized                   Yes
