write_txt_file
Path: CARNOT/misc

Purpose
This functions write data to a text file.

Description
This function is similar to the simulink block "to file", but here the data are not saved as matlab file (binary) but the data is written to an ascii-txt-file, which can be read by an usual editor. It’s also possible to control with the enable signal, when the data is written. Another difference is, that it can be chosen, if the data is appended to an existing file or if a new file is created.

Inputs:

  • input data
  • enable signal [0 or 1]

Parameters and Dialog Box
