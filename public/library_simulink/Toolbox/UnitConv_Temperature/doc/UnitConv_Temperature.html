unitconv_temperature
Path: CARNOT/misc

Purpose
This block converts a temperature form a unit to another. This function also exists in a m-function.

Description
The availlable units are :
 - Celsius degree
 - Kelvin
 - Farenheit degree.

Warning : a difference of temperature doesn't need to be convert from
Kelvin to Celsius.

Parameters and Dialog Box

 

Characteristics
Direct Feedthrough   Yes
Sample time                Inherited from driving block
Vectorized                   No
