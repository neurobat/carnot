THB2THV

Description

This block THB2THV converts the Thermo Hydraulic Bus (THB) into a Thermo Hydraulic Vector (THV).

Inputs

Input Unit Used for Expected values
THB see THB fluid data -

Outputs

Output Unit
THV see THV

Function

This block converts the Thermo-Hydraulic-Bus to the old Thermo-Hydraulic-Vector from Carnot. The bus elements of the THB are converted to a Vector. The THV consists of 20 elemtens. The first 10 elements are defined in Carnot. The second set of 10 elements is free to be used. In VCarnot e.g. element 11 is the hydraulic inductance. Old Carnot blocks can handle this by ignoring the elements 11 to 20.

The elements of the THB and THB are described in the corresponding sections (see below).
