monoflop_realtime
Path: CARNOT/misc

Purpose
This function creates a trigger signal.

Description
This function creates a signal depending on the input signal, e.g. a periodic pulse. The duration of the new signal does not depend on the simulation time of the model, but on the system time, so it's possible to perform realtime control on connected hardware platforms. So whenever the input signal rises (or falls) the functions generates for the specified time interval a signal of 1. The user can specify, when the signal is generated (rising, falling, both).

Inputs:

  • input signal

Outputs:

  • output signal
