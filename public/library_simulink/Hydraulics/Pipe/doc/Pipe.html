Purpose
To model of pipe connection between two components.

Description

Parameters and Dialog Box


  • diameter of the pipe [m]
  • length of the pipe [m]
  • heat loss coefficient [W/(m2K)] (reference is inner pipe surface)
  • effective axial heat conduction [W/(mK)] ( = A(cross-section of wall) / PI*(d2/4)*lambdaWall)
  • capacity of pipe wall per length [J/(m*K)] (heat capacity = cWall*mWall/length)
  • initial temperature [°C]
  • number of nodes (for numeric, a reasonable value is one node per meter of pipe length)
  • pressure drop linear in mass flow [Pa*s/kg]
  • pressure drop linear in mass flow [Pa*s2/kg2]
  • height between pipe inlet and pipe outlet (outlet above inlet is positive). For simplification you may set height to zero in pump systems when thermosyphon flow is not significant.
