Pipe
Path: CARNOT/Storage/Thermal/Storage_Ports

Location:


Purpose
Model of a simple pipe ending of the storage wall.

Description
This block is a simple pipe ending at the wall of the storage. The fluid from this pipe is released directly into the storage.
The pipe connection inputs are:

  • the corresponding port-output "from_storeT" of the storage.
  • the Thermo-Hydraulic Bus from the inport

The outputs are:

  • "to_storeT" connection vector as input for the storage_Tnodes block
  • the Thermo-Hydraulic Bus as outport of the subsystem "storage"

When creating connections, copy the block and connect the THBin and THBout to the corresponding in- and outlet ports. Then connect "from_storeT" of the pipe to "port n" of the "sfun_storage_Tnodes" block. Connect "Tnodes" of the "sfun_storage_Tnodes" to "from_storeT" of the pipe. 
The block transforms the incoming THB from a pipe or a pump to a vector which the storage interprets as heat exchanger.

Stratified Charging or Discharing (Pipe_stratified)

The model is an ideal stratified charging or discharing device. The massflow is entering the storage at a node between the inlet and the outlet depending on the storage temperature. The last node at the outlet is always charged/discharged by the entering massflow.

 Stratified charging (inlet above outlet): The model starts the check at the entry node defined by the relative hight. The check is going down in the storage until it reaches a node where the temperature is lower than the entering temperature. At this node it starts charging the storage. When the temperature of the storage is higher till the outlet node, the outlet node is taken as inlet node for the massflow.
 To model a complete stratified charging heat exchanger, set inlet-position to 1 and outlet- position to 0. The port also models stratified discharging for inlet below outlet. In this case the port starts discharging the storage if the flow temperature reaches a node where the temperature is higher.

Stratified dischaging (outlet above inlet):   The check is going upwards until the outlet node is reached. The entering massflow is set to zero if the temperature in the storage is below the inlet temperature of the pipe.

Parameters and Dialog Box


relative height of inlet [(0 = bottom) ... (1 = top)]
relative height of outlet [(0 = bottom) ... (1 = top)]
