sensitive room temperature
Path: CARNOT/outputs

Purpose
Calculation of sensitive temperature from noode temperature

Description
This block merges the convective and radiative zone node temperature to the SENSITIVE ZONE TEMPERATURE (weighted arithmetic average of both components).

[T] = [Tc; Tr]:

temperature vector of node, [°C]

Ts:

sensitive zone temperature, [°C]

Parameters and Dialog Box

Characteristics
Direct Feedthrough   Yes
Sample time                Inherited from driving block
States                           corresponding to the number of nodes
Vectorized                   No
