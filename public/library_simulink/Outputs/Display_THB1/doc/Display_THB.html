display_THV
Path: CARNOT/outputs

Purpose
The block shows the elements in the Thermo-Hydraulic Vectors.

Description
The block shows the temperature, mass flow rate, pressure, fluid type, fluid mixture or diameter of 6, 4, 2 or 1 Thermo-Hydraulic-Vector. In the parameter block the physical value can be chosen. The number of inputs can be changed by changing the Mux- and Demux-blocks in the subsystem. Be sure to short all unconnected outputs to the Terminator block to avoid a warning in the MATLAB command window. In SIMULINK version 2.0 scopes do not open when starting the simulation. You have to double- click on the scope to see the graph.

Example: see storage demos of CARNOT
