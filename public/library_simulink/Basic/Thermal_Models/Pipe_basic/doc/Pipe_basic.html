Pipe_basic
Path: CARNOT/Basic/hydraulics

Purpose
To model a pipe connection between two components including consideration of the wall capacity and other transfer mechanisms.

Description
The pipe is a one-dimensional mulitnode model. It includes heat capacity of the wall, heat losses to the environment, thermal conductivity between nodes and heat transfer by mass flow. The basic differential equation for the heat and mass transfer is solved by the finite volume method (Patankar, 1980). The transfer due to convection is calculated by the upwind scheme (the flow only comes from the last node in flow direction). Thermal conductivity is calculated by central differences. The resulting differential equation for one node is:

       (cwall*length/Vnode) * dT/dt = U * Aloss / Vnode * (Tamb - Tnode) + cond / dh2 * (Tnextnode - Tnode) + cond / dh2 * (Tlastnode - Tnode) + mdot * cp / Vnode * (Tlastnode - Tnode)

Parameters and Dialog Box

  • inner diameter of the pipe [m]
  • length [m]
  • heat loss coefficient in [W/(m2K)] (reference is inner pipe surface)
  • effective axial heat conduction [W/(mK)] ( = A(cross-section of wall) / PI*(d2/4)*lambdaWall)
  • heat capacity of pipe per length [J/(mK)]
  • initial temperature [°C]
  • number of nodes (for numerics, a reasonable value is one node per meter of pipe length)
