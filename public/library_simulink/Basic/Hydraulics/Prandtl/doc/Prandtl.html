prandtl
Path: CARNOT/basics

Purpose
This block calculates the Prandtl number.

Description
The block calculates the term Pr, known as the Prandtl number.

Pr = cp*h/l

cp - heat capacity
h - viscosity
l - thermal conductivity

The calculation is carried out in the S-function prandtl which is part of the Carlib Library depending on the fluid mix, fluid type, temperature T and pressure p. The material properties functions can also be called directly from the command window by typing

      prandtl(temperature, pressure, fluid_ID, fluidmix)

Instead of the italic variables the actual value of the respective property has to be entered. This is possible for the Carlib functions density, thermal_conductivity, heat_capacity, kinematic_viscosity, prandtl.
For more information about the function type

      help prandtl

to the workspace as for any other MATLAB function.
