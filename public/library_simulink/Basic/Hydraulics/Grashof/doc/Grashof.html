Purpose
The block calculates the Grashof number.

Description
The block calculates the Grashof number according to the

Inputs:

  • wall temperature Twall [°C]
  • temperature in the infinity Tinf [°C]
  • pressure [Pa]
  • fluid_ID (see below)
  • fluid_mix percentage
  • dimension (characteristic length of the system)

The block calls the function "grashof" in the CARNOT Carlib Library where the calculation is carried out. The material properties functions can also be called directly from the command window by typing

      grashof(Twall, Tinf, pressure, fluidtype, fluidmix)

Instead of the italic variables the actual value of the respective property has to be entered. This is possible for the Carlib functions density, thermal_conductivity, heat_capacity, kinematic_viscosity, prandtl, vapourpressure and saturationtemperature. For more information about the function type

      help grashof
