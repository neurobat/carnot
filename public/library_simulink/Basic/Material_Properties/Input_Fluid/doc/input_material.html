input_<material>
Path: CARNOT/materials

Purpose
The block facilitates the input of the material functions

Description
The Input block provides a mask in which the inputs for the calculation of a fluid property_ID like the density, viscosity or the enthalpy can be entered easily . The definition of the fluid mix parameter especially becomes more clear for the different fluids. The blocks exist for the materials water, air and Typhocor L (mixture water-glycol). They are intended to work with the material functions. In case of
water: the fluid mix defines the vapour content of the fluid [0..1] if the water is in the two phase area
air: the fluid mix defines the absolute water content [kgwater/kgair] of the moist air water-glycol mixture: the fluid mix defines the volume content [0..1] of glycol

Parameters and Dialog Box

  • temperature in degree Celsius
  • pressure in [Pa]
  • fluid mix (see above)

Characteristics
Sample time         Inherited from driving block
