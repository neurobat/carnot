thermal conductivity
Path: CARNOT/materials

Purpose
Calculates the thermal conductivity of the specified material.

Description
Calculates the thermal conductivity in [W/(m*K)] according to the

Inputs:

  • temperature[°C]
  • pressure
  • fluid types
  • fluid mix [0..1]

The block calls the function "heat_conductivity" in the CARNOT Carlib Library where the calculation is carried out.

The material properties functions can also be called directly from the command window by typing

      thermal_conductivity(temperature, pressure, fluidtype, fluidmix)

Instead of the italic variables the actual value of the respective property has to be entered. This is possible for the Carlib functions density, thermal_conductivity, heat_capacity, kinematic_viscosity, prandtl, vapourpressure, saturationtemperature, enthalpy, entropy, evaporation_enthalpy and specific_volume.

For more information about the function type

      help thermal_conductivity

to the workspace like you do with any other MATLAB function.
