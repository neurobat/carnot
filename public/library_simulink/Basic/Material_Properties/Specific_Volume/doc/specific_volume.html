specific volume
Path: CARNOT/materials

Purpose
Calculates the specific volume of the specified material.

Description
Calculates the specific_volume [m3/kg] according to the

Inputs:

  • temperature in [°C]
  • pressure in [Pa]
  • fluid types see above definition
  • fluid-mix [Unit dependents on fluid_type, see below]

The block calls the function "specific_volume" in the CARNOT Carlib Library where the calculation is carried out. The material properties functions can also be called directly from the command window by typing

      specific_volume(temperature,pressure, fluidtype,fluidmix)

Instead of the italic variables the actual value of the respective property has to be entered. This is possible for the Carlib functions density, thermal_conductivity, heat_capacity, kinematic_viscosity, prandtl, vapourpressure, saturationtemperature, enthalpy, entropy, evaporation_enthalpy and specific_volume.
For more information about the function type

      help specific_volume

to the workspace as for any other MATLAB function.
For the vaporous phase the valid temperature range is up to 800 °C and the valid pressure range is up to 49.55 MPa.

For the liquid phase the valid temperature range is between 10 °C and 200 °C and the pressure is limited up to 2 MPa.
