enthalpy
Path: CARNOT/materials

Purpose
This block calculates the specific enthalpy [J/kg] of the specified material.


Description This block calculates the specific enthalpy in [J/kg] according to the

Inputs:

  • temperature[°C]
  • pressure
  • fluid types see above definition
  • fluid mix percentage [0 ..1]

The block calls up the function "enthalpy" in the CARNOT Carlib Library where the calculation is carried out. The material properties functions can also be called directly from the command window by typing

      enthalpy(temperature, pressure,fluid_type,fluid_mix)

Instead of the italic variables the actual value of the respective property has to be entered. This is possible for the Carlib functions density, thermal conductivity, heat capacity, kinematic viscosity, prandtl number, vapour-pressure, saturation-temperature, enthalpy, entropy, evaporation-enthalpy and specific volume. For more information about the function type

      help enthalpy

to the workspace as for any other MATLAB function.

For more information of the fluid types see above definition

For the calculation of steam, the properties are calculated according to the "E.Schmidt: Properties of water and steam". The steam properties can be calculated for a temperature range up to 800°C and a pressure range up to 49.055 MPa.
The calculations of the liquid phase and the saturation states are performed with the polynomials from "B.Glück: Stoffwerte und Zustandsgrößen". The valid temperature range is up to 200°C and the pressure range is up to 2 MPa. The enthalpy of the liquid phase considers only the dependency of the temperature; the pressure dependency is neglected.

The enthalpy is only available for the fluids water (fluid type 1) and air (fluid type 2)
