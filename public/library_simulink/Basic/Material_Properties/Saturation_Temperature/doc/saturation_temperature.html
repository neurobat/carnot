saturationtemperature
Path: CARNOT/materials

Purpose
Calculates the saturation temperature of the moist air at the specified conditions

Description
Calculates the saturation temperature of moist air (fluid_ID 2, see below) or pure water (fluid_ID 1, see below)in [degrees] according to the

Inputs:

  • temperature in [°C]
  • pressure in [Pa]
  • fluid types see above definition
  • fluid-mix [Unit dependents on fluid_type, see below]

The block calls the function "saturationtemperature" in the CARNOT Carlib Library where the calculation is carried out. The material properties functions can also be called directly from the command window by typing

      saturationtemperature(temperature, pressure, fluidtype, fluidmix) Instead of the italic variables the actual value of the respective property has to be entered. This is possible for the Carlib functions density, thermal_conductivity, heat_capacity, kinematic_viscosity, prandtl, vapourpressure, saturationtemperature, enthalpy, entropy, evaporation_enthalpy and specific_volume. For more information about the function type

      help saturationtemperature

to the workspace as for any other MATLAB function.
In the case of pure water (fluid_ID 1) the function saturationtemperature calculates the temperature of saturated steam at the specified pressure.
In the case of moist air (fluid_ID 2) the function saturationtemperature calculates the dew point temperature of the water in the moist air.
See definition of the fluid types above The property is specified by entering a property_ID that is defined by the following table
Definition of the property_ID types
In the same way the phase ID has to be entered in the mask of the block to determine which saturationproperty, water or steam, should be calculated. The phase_ID is defined above.

The valid range of the saturationtemperature is below the critical pressure (p < pcrit= 221,2 bar)

NOTE: At the moment the saturation properties are only available for water
