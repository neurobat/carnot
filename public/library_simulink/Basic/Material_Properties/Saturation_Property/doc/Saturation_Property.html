saturationproperty
Path: CARNOT/materials

Purpose
The block calculates the saturation properties at the specified conditions.

Description
The block calculates the saturation properties of the (') state and the (") state, i.e. the property of boiling fluid or saturated steam according to the equation

prop(x) = x*prop" + (1-x)*prop'

with

prop

specified property

x

vapurcontent

prop'

boiling fluid state

prop"

saturated steam state

Inputs:

  • temperature in [°C]
  • pressure in [Pa]
  • fluid types see above definition
  • fluid-mix [Unit dependents on fluid_type, see definition above]

The block calls the function "satuprop" in the CARNOT Carlib Library where the calculation is carried out. The material properties functions can also be called directly from the command window by typing

      saturationproperty(temperature, pressure, fluidtype, fluidmix, propertyvalue)

Instead of the italic variables the actual value of the respective property has to be entered. This is possible for the Carlib functions density, thermal_conductivity, heat_capacity, kinematic_viscosity, prandtl, vapourpressure, saturationtemperature, enthalpy, entropy, evaporation_enthalpy, specific_volume. For more information about the function type help saturationproperty to the workspace as for any other MATLAB function.
In the case of pure water (fluid ID 1) the function saturationtemperature calculates the values of saturated steam.
In the case of moist air (fluid ID 2) the function saturationtemperature calculates the partial pressure of the water in the moist air.
