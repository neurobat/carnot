vapourpressure
Path: CARNOT/materials

Purpose
Calculates the vapour-pressure of the specified material.

Description
Calculates the vapour-pressure [Pa] of saturated steam according to the inputs:

  • temperature in [°C]
  • pressure in [Pa]
  • fluid types see above definition
  • fluid-mix [Unit dependents on fluid_type, see below]

The block calls the function "vapourpressure" in the CARNOT Carlib Library where the calculation is carried out.
The material properties functions can also be called directly from the command window by typing

      vapourpressure(temperature, pressure, fluidtype, fluidmix)

Instead of the italic variables the actual value of the respective property has to be entered. This is possible for the Carlib functions density, thermal_conductivity, heat_capacity, kinematic_viscosity, prandtl, vapourpressure, saturationtemperature, enthalpy, entropy, evaporation_enthalpy and specific_volume.

For more information about the function type

      help vapourpressure

to the workspace as for any other MATLAB function.
In the case of pure water (fluid ID 1) the function vapourpressure calculates the values of saturated steam.
In the case of moist air (fluid ID 2) the function vapourpressure calculates the partial pressure of the water in the moist air.

The vapourpressure is defined between the triple point at 612 Pa and the critical point at 22.12 MPa
