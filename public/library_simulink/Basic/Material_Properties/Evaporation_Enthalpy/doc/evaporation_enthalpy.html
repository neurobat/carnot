evaporation enthalpy
Path: CARNOT/materials

Purpose
This block calculates the evaporation enthalpy [J/kg] of the specified material.

Description
This block calculates the specific evaporation enthalpy r in [J/kg] according to the

Inputs:

  • temperature[°C]
  • pressure
  • fluid types see above definition
  • fluid mix percentage [0 ..1]

The evaporation enthalpy r is the enthalpy that is necessary to evaporate the material. It is calculated by the difference of the saturation properties of liquid (' state) and vapourous phase (" state) at the saturation point. At the saturation point, the liquid phase is boiling fluid and the vapourous phase is saturated steam:
The block calls the function "evaporation enthalpy" in the CARNOT Carlib Library where the calculation is carried out. The material properties functions can also be called directly from the command window by typing

      evaporation_enthalpy(temperature, pressure, fluidtype, fluidmix)

Instead of the italic variables the actual value of the respective property has to be entered. This is possible for the Carlib functions density, thermal_conductivity, heat_capacity, kinematic_viscosity, prandtl, vapourpressure, saturationtemperature, enthalpy, entropy, evaporation_enthalpy and specific_volume.

      help evaporation_enthalpy

to the workspace as for any other MATLAB function.
The calculation of the saturation states are performed with the polynomials from " Bernd Glück: Zustands- und Stoffwerte."

As the saturationvalues only exist for water the evaporation_enthalpy is also only available for the fluid water (fluid_type 1)

The valid temperature range is up to from the temperature 10°C up to 370°C
