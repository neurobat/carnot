Path: CARNOT/control

Purpose
Hysteresis controller for thermal systems.

Description
The block is a simple hysteresis controller. The outlet is set to:

  • 1if (Thot - Tcold) < deltaTon
  • 0if (Thot - Tcold) < deltaToff

The output is set to zero if the Thot or Tcold is bigger than Tmax

  • Input
  • Thot
  • Tcold

Output
control signal (0 or 1)

Parameters and Dialog Box

If the temperature Thot or Tcold exeeds Tmax, the controler will switch off the control signal.

Characteristics
Direct Feedthrough   Yes
Sample time                Inherited from driving block
