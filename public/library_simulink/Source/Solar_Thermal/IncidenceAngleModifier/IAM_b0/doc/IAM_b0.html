Path: CARNOT/Source/Solar_Thermal/IncidenceAngleModifier

Purpose

USE IN THE "Collector_EN12975_FlatPlate" MODEL.
This block transforms incoming direct and diffuse radiation to an absorbed solar power per unit area.

Description
The direct radiation incidence angle modifier (Kdir) uses the modified b0 equations for flat plate collectors equations from [Isakson 1995].
For direct radiation:
    0° < teta < 60°
        Kdir = 1 - b0 * (1/cos(teta) -1)
     60° <= teta < 90°
        Kdir = (1 - b0) * (1 - (teta-60)/30)
    teta >= 90°
        Kdir = 0
For diffuse radiation:
     Kdfu = constant (teta = 50° incidence angle)
With:
    teta: incidence angle (0° = vertical)
    Kdir: direct radiation incidence angle modifier
    Kdfu: diffuse radiation incidence angle modifier

The Fin Efficiency * Transmittance * Absorbtance "FTA" or F'(TauAlfa) is defined from the incidence angle modifiers Kdir and Kdfu:
    FTA = eta0 / (Kdir(15°)*0.85 + Kdfu*0.15)

Inputs:

  • solar radiation (direct, diffuse) on the collector plane
  • longitudinal and transversal incidence angles of the solar radiation
