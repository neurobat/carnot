THBCreator

Description

The block THBCreator creates a THB.

Inputs

Input Unit

Temperature °C
MassFlow kg/s
Pressure Pa
FluidType 1
FluidMix 1
DiameterLastPiece m
DPConstant Pa

Output Unit

Function

With this block a THB is created. The THB should be used in VCarnot instead of the old Carnot THV. It can be choosen if a certain element of the THB is entered in the mask or as an inport. Furthermore, the values entered in the mask can be used for the block annotation.
