weather from file
Path: CARNOT/weather

Purpose
Input block for external weather data files.

Description
This block corresponds to the simulink block "From File". You can read in weather data from external files, like TRY (Test Reference Years) The data files have to be in a Matlab readable form, i.e. in the first column the time has to appear. To bring Test Reference Years in the Weather data format of CARNOT the m-file Wprepro.m, which is located in the CARNOT root directory can be used. The m-files Meteonorm2wformat.m and MeteonormMinute2wformat.m can be used for Meteonorm files. To get more information about the utility functions you can type

      help write_try

to the workspace, like as for any other MATLAB function. For weather input data files of a different format the Wprepro.m has to be adapted to the suitable format.

Parameters and Dialog Box

Characteristics
Direct Feedthrough    Yes
Sample time specify   -1 (although weather data usually have 3600 s)
Vectorized                   No
