valve_thermostatic
Path: CARNOT/hydraulics

Purpose
Regulation element for room heating system.

Description
The thermostatic valve is used to reduce the mass flow in a space heating system when temperature of the sensor reaches its desired value. The characteristics of a typical valve are used for the pressure drop. The temperature of the sensor is delayed by the heat capacity from the incoming temperature. The position of the valve is calculated by
Position = -Slope * (Tsensor-Tset)
The working point is the position of the valve if the sensor temperature is the same as the set temperature.

Inputs:

  • THV inlet
  • Temperature for the sensor

Output:

  • THV outlet
