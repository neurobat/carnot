readfile
Path: CARNOT/basics

Purpose
This block reads a matrix or variables from a file.

Description
For this function it is important, if it depends on the time or not. In the case of time dependence a matrix is read with the given number of column, where the first value is the time variable. This time is compared to the simulation time, so data, which are not in the matrix, will be linear interpolated. Comment lines have to begin with % or //.
When the simulation time exceeds the time in the file, the function reads the file from the beginning.
In case of independence of time all variables (not a matrix) in the file are read in the first function call. Then at every call the same variables will be put out.
It is important to give the right number of rows or variables as parameter.
This function can be used to read different data files, when the simulink function is compiled with the Real Time Workshop, because the data isn’t written in the RTW-executable, but is read from a file.
Outputs:

  • when depending on time: matrix row without time
  • when not depending on time: all variables in the file

Parameters:

  • number of matrix columns (including time column) when timedependent or number of all variables when not timedependent
  • flag of time dependence
  • filename
