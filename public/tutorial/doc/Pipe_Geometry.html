pipe
Path: CARNOT/hydraulics

Purpose
To model a pipe connection between two components including consideration of the wall capacity and other transfer mechanisms.

Description
For all thermal calculations see block "Pipe_basic". The only difference is the pressure drop calculation.

This function also calculates the pressure drop in the pipe according to VDI Wärmeatlas, 1994


Example: see parallel.mdl

Literature: Patankar, 1980

Parameters and Dialog Box

  • inner diameter of the pipe [m]
  • length [m]
  • heat loss coefficient in [W/(m2K)] (reference is inner pipe surface)
  • effective axial heat conduction [W/(mK)] ( = A(cross-section of wall) / PI*(d2/4)*lambdaWall)
  • heat capacity of pipe per length [J/(mK)]
  • initial temperature [°C]
  • number of nodes (for numerics, a reasonable value is one node per meter of pipe length)
  • wall roughness
  • sum of bends (used for pressure drop), set 1 for a 90° bend, 0.667 for a 45° bend
  • height between pipe inlet and pipe outlet (outlet above inlet is positive). For simplification you
  • may set height to zero in pump systems when thermosyphon flow is not significant.

Characteristics
Direct Feedthrough   Yes
Sample time                Inherited from driving block
States                           corresponding to the number of nodes
Vectorized                   No
