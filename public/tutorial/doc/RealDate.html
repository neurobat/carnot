Real Date
Path: CARNOT/Toolbox

Purpose
This functions puts out the system time in seconds.
Output 0 s means January, 1st 0:00

Parameters
Selector:
    Year - current year
    Month - current month
    Day - current day in the month (date)
    Hour - current hour (time)
    Minute - current minute (of the time)
    Second - current second (of the time)  

 

Outputs:

  • current date or time
