x2rel_hum
Path: CARNOT/basics

Purpose
The function converts the absolute humidity, i.e. the load of water in moist air, to the relative humidity.

Description
The function converts the relative humidity to the absolute humidity, i.e the load of water in moist air according to the inputs:

  • temperature [°C]
  • pressure [Pa]
  • absolute humidity [kgwater/kgair]

Attention: This block is only to be used in the case of moist air (fluid type 2)

Characteristics
Direct Feedthrough   Yes
Sample time                Inherited from driving block
Vectorized                   No
