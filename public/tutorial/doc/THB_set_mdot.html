create THV
Path: CARNOT/misc

Purpose
Creates a Thermo-Hydraulic Vector from the inputs temperature and mass flow rate.

Description
This block is used to enter measured data like massflow and temperature in the THV. All other values are set in the dialog box. The output of the block is a Thermo-Hydraulic Vector.

Inputs:

  • temperature [°C]
  • mass flow rate [kg/s]

Outputs:

  • THV

Parameters and Dialog Box

Characteristics
Direct Feedthrough   Yes
Sample time                Inherited from driving block
Vectorized                   No
