pump const
Path: CARNOT/Pumps

Purpose
Very simple pump model where the mass flow rate is set to maximum value.

Description
This is a very simple pump, where the mass flow is set to the maximum mass flow rate specified in the mask. The direct feedthrough in the temperature is broken by a transfer function with a time-constant of 100 s. The pump has a heat capacity of 1 kg steel, so the temperature at the outlet is not directly equal to the inlet temperature.

Inputs:

  • THV
  • control signal, limited to the range [0..1]

Outputs:

  • THV

NOTE: If you use free components of the THV, you have to change the Pumps. To avoid algebraic loops, unused components of the THV are not fed through.

Example: the load in storage demos

Parameters and Dialog Box

  • maximum mass flow rate [kg/s]
  • the fluid
  • percentage of fluid mix (only for mixtures like water - glycol)
  • inlet and outlet diameter (used for pressure drop calculation in the following pipe)
  • system pressure [Pa]
  • level of pressure calculation [only friction, friction and thermal head]

Characteristics
Direct Feedthrough   No
Sample time                Inherited from driving block
Vectorized                   No
