Introduction

The CARNOT is a toolbox extension for MATLAB SIMULINK. It is a tool for the calculation and simulation of the thermal components of heating systems with regards to conventional and regenerative elements.
The CARNOT Toolbox is a libraries of typical components of these systems. It is organized in Blocksets like the SIMULINK Library itself. The handling of the blocks is exactly the same as in SIMULINK, so that users familiar with SIMULINK can directly use the new Blocksets in the same way.

  • the CARNOT Library contains a wide range of elements which can be user-defined and some basic functions to facilitate the modelling. The CARNOT Library was designed for sophisticated simulation and analysis of heating systems.
