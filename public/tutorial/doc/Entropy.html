entropy
Path: CARNOT/materials

Purpose
This block calculates the specific entropy [J/(kg K)] of the specified material.

Description
This block calculates the specific entropy in [J/(kg K)] according to the

Inputs:

  • temperature[°C]
  • pressure
  • fluid types see above definition
  • fluid mix percentage [0 ..1]

The block calls the function "entropy" in the CARNOT Carlib Library where the calculation is carried out. The material properties functions can also be called directly from the command window by typing

      entropy(temperature,pressure, fluidtype,fluidmix)

Instead of the italic variables the acual value of the respective property has to be entered. This is possible for the Carlib functions density, thermal_conductivity, heat_capacity, kinematic_viscosity, prandtl, vapourpressure, saturationtemperature, enthalpy, entropy, evaporation_enthalpy and specific_volume. For more information about the function type

      help entropy

to the workspace as for any other MATLAB function.

For the calculation of steam, the properties are calculated according to the " E.Schmidt". The steam properties can be calculated in a temperature range up to 800°C and a pressure range up to 49.055 MPa. The calculations of the saturation states are performed with the polynomials fitted from "E.Schmidt: Properties of water and steam". The valid temperature range is from the triple temperature 0.01°C up to the critical temperature Tk = 374.15°C and the pressure range respectively from the triple pressure 612 Pa up to 22.1287 MPa. The entropy values for the liquid phase are not yet implemented.

The entropy is only available for the fluids water (fluid type 1) and air (fluid type 2)
