weather_2_THV
Path: CARNOT/weather

Purpose
The block transfers the incoming weather data to a Thermo-Hydraulic Vector.

Description
This block provides the input THV to any component using ambient air like ventilation systems or the heat pump. Only the ambient temperature is used in the THV. The other input data is sent to a Terminator block. Further data required in the THV is constant.

Parameters and Dialog Box

Characteristics
Direct Feedthrough   Yes
Sample time                Inherited from driving block
Vectorized                   No
