THV2THB

Description

This block THV2THB converts the Thermo Hydraulic Vector (THV) into a Thermo Hydraulic Bus (THB).

Inputs

Input Unit Used for Expected values
THV see THV fluid data -

Outputs

Output Unit
THB see THB

Function

This block converts the old Thermo-Hydraulic-Vector to the Thermo-Hydraulic-Bus. The THV is used by Carnot. VCarnot shall use the THB. The block is needed to use old Carnot blocks within VCarnot. The THV has exactly 20 elements, whereas the first ten are used by Carnot. The latter ten elements can be used freely - they are ignored in Carnot. In VCarnot e.g. element 11 is used for the hydraulic inductance. The exact construction of the THB and the THV are described in the correspondig help pages (see links in the end of this page).
