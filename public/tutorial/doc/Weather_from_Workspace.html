weather from Workspace
Path: CARNOT/weather

Purpose
Input block for external weather data from workspace.

Description
This block corresponds to the simulink block "From Workspace". You can read in weather data from external files, like TRY (Test Reference Years) The data files have to be in a Matlab readable form, i.e. in the first column the time has to appear. To bring Test Reference Years in the Weather data format of CARNOT the m-file Wprepro.m, which is located in the CARNOT root directory can be used. To get more information about the utility functions you can type

      help write_try

to the workspace, like as for any other MATLAB function. For Weather input data files of a different format the Wprepro.m has to be adapted to the suitable format. Before the data is available it has to be loaded onto the Workspace in the way that is described in chapter 2.3.

If you are not intend to make operation on data before you use it in the model, prefer using weather_from_file.

Parameters and Dialog Box

Characteristics
Direct Feedthrough    Yes
Sample time specify   -1 (although weather data usually have 3600 s)
Vectorized                   No
