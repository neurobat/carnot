velocity
Path: CARNOT/basics

Purpose
Calculates the velocity in a pipe in [m/s] assuming a round pipe. The diameter is taken from the Thermo-Hydraulic Vector.

Description
Calculates the velocity in a pipe according to the inputs:

  • mass flow rate [kg/s]
  • pressure [Pa]
  • fluid types see above definition
  • fluid mix percentage [0 ..1]
  • inner pipe diameter [m]

and the equation

      v = mdot/(density*cross section)

For more information of fluid types see above definition

Characteristics
Direct Feedthrough   Yes
Sample time                Inherited from driving block
Vectorized                   No
