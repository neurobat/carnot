weather simple model
Path: CARNOT/weather

Purpose
Simple model to output the weather data.

Description
The model describes the weather data by using standard input functions for the zenith and azimuth angle and the direct and diffuse radiation. For the direct and the diffuse radiation, a sine wave is used to model the weather conditions. The solar position and incidence angles of the radiation are calculated from the geographical position and the simulation time.

Characteristics
Direct Feedthrough   Yes
Sample time                None
Vectorized                   No
