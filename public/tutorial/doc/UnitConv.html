unit conv / unit conv temperature
Path: CARNOT/misc

Purpose
Converts a value from a specified unit to another unit.

Description
The symbols in a specified unit have to be separated by * or /. So kWh has to be written as kW*h. The common signs are allowed, e.g. kW, mm, µJ.
Beside the units this functions knows some constants, e.g. when using the "unit" c, you get the lightspeed.

It’s also possible to use this function from the workspace by typing the command unitconv: e.g. unitconv(12,’ft/s’,’km/h’,1) The last number 1 is only flag. If this flag is 1, the basic SI-units are also printed out.
This block is also available as workspace function unitconv.
Input:

  • value in old unit

Output:

  • value in new unit

Note:

  • a year is set to 365 days

The Block unitconv_temperature is a special case of unitconv, witch convert only temperatures between the three unity : °C, °F and K.
