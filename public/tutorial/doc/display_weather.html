display weather
Path: CARNOT/outputs

Purpose
The block shows the specified weather variables from the weather vector.

Description
The block shows the specified weather variables from Weather data vector. In the parameter block the weather variables can be chosen. The number of inputs can be changed by changing the Mux- and Demux-blocks in the subsystem. Be sure to short all unconnected outputs to the Terminator block to avoid a warning in the MATLAB command window. In SIMULINK version 2.0 scopes do not open when starting the simulation. You have to double- click on the scope to see the graph.

Example: see storage demos of CARNOT
