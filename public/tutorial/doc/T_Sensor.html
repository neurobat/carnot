measure T
Path: CARNOT/control

Purpose
Surface measurement sensor for temperatures with time constant.

Description
This is the model of a surface temperature sensor. It can be combined with the storages to get the node temperature in degree Celsius. It breaks the algebraic loop which is created when connecting the node temperature directly to the controller.

Input

  • THV

Output

  • THV
  • temperature in [°C]

Parameters and Dialog Box

  • time constant of the sensor [s]
  • delay-time [s]
  • initial temperature of sensor [°C]

Characteristics
Direct Feedthrough   Yes
Sample time                Inherited from driving block
Vectorized                   No
