density
Path: CARNOT/materials

Purpose
Calculates the density of the specified material.

Description
Calculates the density according to the

Inputs:

  • temperature[°C]
  • pressure
  • fluid types see above definition
  • fluid mix percentage [0 ..1]

output:

  • density in [kg/m3]

The block calls the function "density" in the CARNOT Carlib Library where the calculation is carried out. The functions can also be called directly from the command window by typing

      density(temperature, pressure, fluidtype, fluidmix)

Instead of the italic variables, the actual value of the respective property has to be entered. This is possible for the Carlib functions density, thermal_conductivity, heat_capacity, kinematic_viscosity, prandtl, vapourpressure, saturationtemperature, enthalpy, entropy, evaporation_enthalpy and specific_volume.
For more information about the function type

      help density
