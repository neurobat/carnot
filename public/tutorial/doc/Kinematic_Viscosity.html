kinematic_viscosity
Path: CARNOT/materials

Purpose
Calculates the kinematic viscosity of the specified material.

Description
Calculates the kinematic viscosity in [m2/s] according to the inputs:

  • temperature in [°C]
  • pressure in [Pa]
  • fluid type
  • fluid-mix percentage

The block calls the function "viscosity" in the CARNOT Carlib Library where the calculation is carried out. The material properties functions can also be called directly from the command window by typing

      kinematic_viscosity(temperature, pressure, fluidtype, fluidmix)

Instead of the italic variables the actual value of the respective property has to be entered. This is possible for the Carlib functions density, thermal_conductivity, heat_capacity, kinematic_viscosity, prandtl, vapourpressure, saturationtemperature, enthalpy, entropy, evaporation_enthalpy and specific_volume.
For more information about the function type

      help kinematic_viscosity

to the workspace as for any other MATLAB function.
