floor_in
Path: CARNOT/loads/wall and floor/wall_floor

Purpose
Model of an inner floor (or ceiling) of a room.

Description
Convective heat transfer

The program assumes a constant convective heat transfer coefficient. Horizontal ceilings and floors are calculated as follows:

c = 2.0[W/m2]

For the further description of the model see wall_out.

NOTE:In some cases this can result in distorted calculations as heat transfers are temperature- dependent. In addition convective temperature transfer is influenced by the height of vertical plane.

Parameters and Dialog Box

Characteristics
Direct Feedthrough   Yes
Sample time                Inherited from driving block
States                           corresponding to the number of nodes
Vectorized                   No
