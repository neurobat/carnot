*                WELCOME TO THE CARNOT TOOLBOX                          *
*                                                                       *
*************************************************************************


The CARNOT toolbox is an extension for the simulation program 

MATLAB SIMULINK. It is required that you have already installed MATLAB 

SIMULINK on your platform.

                            INSTALLATION


Open "CARNOT_Installation_Guide.txt" for installation instructions.
